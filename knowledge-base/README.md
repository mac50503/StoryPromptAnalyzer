## Vistas Disponibles

Esta base de conocimiento ofrece dos formas de navegar el contenido:

1. **[Vista por RuleFlows](by-ruleflow/README.md)** - Organizada por procesos de negocio (RECOMENDADA)
   - Ideal para entender flujos completos (ej: cómo se calcula el pago de tripulación)
   - Muestra qué funciones orquesta cada RuleFlow
   - Útil para análisis de impacto y debugging

2. **Vista por Entidad** (esta página) - Lista alfabética de todas las funciones y rulesets
   - Ideal para buscar una función específica
   - Referencia completa de todas las entidades
